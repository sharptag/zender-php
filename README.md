# zender-php
<article class="markdown-body entry-content" itemprop="text">

<p>Zender is a free email delivery service, it is built as Web API to serve marketers &amp; developers and make it easy to send email. http://zender.sharptag.com</p>
<p><strong>This library allows you to quickly and easily use the Zender Web API v1 via PHP.</strong></p>
<p>Please browse the rest of this README for further detail. We appreciate your continued support, thank you!</p>
